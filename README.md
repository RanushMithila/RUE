Rue: Melody of the Mist 🌿🐍

A 3D Atmospheric Survival Adventure Developed for IN4630 – Game Theory | University of Moratuwa
📖 Overview

Rue is a tiny creature lost in a massive, hyper-realistic paddy field. This 3D experience focuses on scale, tension, and emotional storytelling. Navigate through mud tunnels, slide through mouse holes, and hide from a relentless Snake predator to find the safety of the "Soda Can Sanctuary."
🎮 Gameplay Mechanics

    3D Micro-Platforming: Precision movement through oversized vegetation.
    Predator AI (Snake): Dynamic chase sequences using Unreal AI Perception.
    The Mouse Hole Slide: High-speed spline-based sliding with obstacles.
    Stealth Set-Pieces: Scripted hiding moments (The Pot and The Branch).
    Lumen & Nanite: Utilizing UE5's lighting for a cinematic "Macro-Photography" look.

🚀 How to Run

    Clone the repo: git clone https://github.com/[YOUR-USERNAME]/RUE.git
    Ensure Git LFS is installed: git lfs install
    Right-click Rue.uproject and select Generate Visual Studio project files (if using C++).
    Open the project in UE5.
